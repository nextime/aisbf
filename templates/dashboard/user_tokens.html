Add the token to every request in the Authorization header:

Authorization: Bearer YOUR_API_TOKEN

Token scopes:

ScopeAccess
apiProxy API endpoints only (/api/u/…)
mcpMCP tool endpoints only (/mcp/u/…)
bothBoth API and MCP endpoints
