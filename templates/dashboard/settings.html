Onion Address:
Expose AISBF over TOR network as a hidden service

⚙️ Required torrc Configuration

Before enabling Tor, you must configure your torrc file (usually /etc/tor/torrc):

Step 1: Generate Password Hash 
tor --hash-password "your_secure_password"
Step 2: Add to torrc

For temporary hidden service (new address each restart):

ControlPort 9051
HashedControlPassword 16:YOUR_HASHED_PASSWORD_HERE

⚠️ Leave "Hidden Service Directory" field blank below for ephemeral service

For permanent hidden service (same address across restarts):

ControlPort 9051
HashedControlPassword 16:YOUR_HASHED_PASSWORD_HERE

HiddenServiceDir /home/yourusername/.aisbf/tor_hidden_service
HiddenServicePort 80 127.0.0.1:{{ config.server.port if config.server else 17765 }}

⚠️ Replace /home/yourusername with your actual home directory path (torrc does NOT expand ~)

Step 3: Restart Tor 
sudo systemctl restart tor  # Linux
brew services restart tor   # macOS
